# interStella 기획서

## 문서 운영 규칙
- 이 문서는 Append-Only로 운영한다.
- 기존 기록은 삭제하지 않는다.
- 수정이 필요하면 기존 내용을 지우지 말고 `변경 이력`에 새 항목으로 추가한다.

## 프로젝트 개요 (기준선)
- 장르: 물리 기반 협동 우주 멀티플레이
- 엔진/플랫폼: Unity 2022, Steam 멀티플레이
- MVP 인원: 2인 코옵 (4인 확장 고려 설계)
- 이동 모델: 무중력 6DoF
- 핵심 자원: 우주복 연료

## 코어 재미 축
- 물리 기반 이동 자체가 재미의 중심
- 플레이어 간 연결 제약(테더)으로 협동 의사결정과 사고 유발
- 수집 -> 운반 -> 수리의 반복 가능한 루프

## MVP 목표 루프
- 세션 참가
- 플레이어 스폰
- 연료를 쓰며 이동
- 테더 제약 하 협동 이동
- 스크랩 수집/운반
- 정거장 수리 진행
- 성공 또는 실패 상태를 명확하게 종료

## 네트워크 원칙 (MVP)
- 호스트 권한(Host-authoritative) 기준
- 입력은 로컬, 최종 게임 진실은 호스트
- Late Join 미지원 (MVP)
- Host Migration 미지원 (MVP)
